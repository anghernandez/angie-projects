# Battle City in C

This is a Battle City game implemented in the C programming language using SDL2 and SDL2_image.
The project recreates the classic gameplay style of Battle City, where the player controls a tank, defeats enemy tanks, and protects the base.

## Project Overview

The game includes:

- 5 fully functional levels
- 6 enemy types distributed across different levels
- Player lives system
- Score system
- Power-ups
- Game states such as menu, gameplay, victory, and game over
- Destructible and indestructible map elements
